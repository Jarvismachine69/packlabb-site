How to export this as a real logo file

  1. Screenshot method (quickest): Right-click any logo above → "Inspect" → find the element → right-click in DevTools → "Capture node screenshot". Done, PNG saved.
  2. Full page screenshot: In Chrome, press F12 → Ctrl+Shift+P → type screenshot → "Capture full size screenshot".
  3. For a proper vector file (AI/SVG): You'll want to open this in Figma — paste the SVG code below and export from there. Or use a designer to finalise it with outlined fonts so it works everywhere.
  4. Figma tip: Go to figma.com → New file → paste the SVG icon code → set font to Inter 900 → type "pack" in black, "labb" in #3B82F6.
